## Description
This is a **command-line Python application** that calculates the BMI of a user based on their weight and height.  
It then classifies the BMI into **Underweight, Normal weight, Overweight, or Obese** categories.

The project is suitable for **beginners learning Python** and covers concepts such as:
- Functions
- Conditional statements
- User input validation
- Error handling

---

## Features
- Prompt user to input weight (kg) and height (m)
- Calculate BMI using the formula:  
  `BMI = weight / (height ** 2)`
- Categorize BMI into health categories
- Handle invalid or negative inputs gracefully
